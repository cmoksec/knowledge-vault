Методы Path

| Метод | Описание | Пример использования |
| --- | --- | --- |
| `Path(path_string)` | Создает объект Path, представляющий указанный путь. | `path = Path('/path/to/file.txt')` |
| `.resolve()` | Возвращает абсолютный путь, разрешая все символические ссылки. | `absolute_path = path.resolve()` |
| `.exists()` | Проверяет, существует ли файл или каталог. | `if path.exists(): ...` |
| `.is_file()` | Проверяет, является ли путь файлом. | `if path.is_file(): ...` |
| `.is_dir()` | Проверяет, является ли путь каталогом. | `if path.is_dir(): ...` |
| `.parent` | Возвращает объект Path, представляющий родительский каталог. | `parent_dir = path.parent` |
| `.name` | Возвращает имя файла или каталога. | `file_name = path.name` |
| `.suffix` | Возвращает расширение файла. | `file_extension = path.suffix` |
| `.stem` | Возвращает имя файла без расширения. | `file_stem = path.stem` |
| `.joinpath(other)` | Объединяет текущий путь с другим путем. | `new_path = path.joinpath('another_file.txt')` |
| `.mkdir(parents=False, exist_ok=False)` | Создает каталог. | `path.mkdir(parents=True, exist_ok=True)` |
| `.touch()` | Создает пустой файл. | `path.touch()` |
| `.read_text(encoding=None)` | Читает содержимое файла как строку. | `content = path.read_text(encoding='utf-8')` |
| `.write_text(data, encoding=None)` | Записывает строку в файл. | `path.write_text('some text', encoding='utf-8')` |
| `.iterdir()` | Возвращает итератор по содержимому каталога. | `for item in path.iterdir(): ...` |
| `.glob(pattern)` | Возвращает итератор по путям, соответствующим шаблону. | `for file in path.glob('*.txt'): ...` |
| `.rename(target)` | Переименовывает файл или каталог. | `path.rename(new_path)` |
| `.replace(target)` | Перемещает или переименовывает файл или каталог. | `path.replace(new_path)` |
| `.unlink()` | Удаляет файл. | `path.unlink()` |
| `.rmdir()` | Удаляет каталог (если он пуст). | `path.rmdir()` |
| `.with_name(name)` | Возвращает новый путь с измененным именем. | `new_path = path.with_name('new_name.txt')` |
| `.with_suffix(suffix)` | Возвращает новый путь с измененным суффиксом. | `new_path = path.with_suffix('.csv')` |
